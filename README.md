### ColorfulChalk

placeholder readme

requires [Socks](https://thunderstore.io/c/webfishing/p/toes/Socks/) and [Lure Refreshed](https://thunderstore.io/c/webfishing/p/TeamLure/LureRefreshed/)

to select a color, press P to open up the color picker. alternatively, you can enter !color (hex value, without #) in global chat.

enter !save in global chat to save a copy of the current color atlas

thanks for A-10 for creating the sparkly chalk sprite!
